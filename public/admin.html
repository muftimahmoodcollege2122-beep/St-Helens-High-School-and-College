📥 Bulk Import Results

📋 How to use:
1. Click "⬇ Download Template" to get the sample Excel/CSV file.
2. Fill in student results — one student per row.
3. Subject columns: use SubjectName_Total and SubjectName_Obtained (e.g. Urdu_Total, Urdu_Obtained).
4. Or simply put subject names as column headers with marks as values (total assumed 100).
5. Select the exam, year, upload the CSV and click Import.
Supported: .csv files. For Excel, save as CSV first (File → Save As → CSV).

📥 Bulk Import Students

📋 How to use:
1. Click "⬇ Download Template" to get the sample CSV.
2. Fill student data — one row per student.
3. Required columns: rollNo, name, class, gender.
4. Optional: fatherName, phone, fatherPhone, section, address, status.
5. Upload the CSV and click Import. Supports 5000+ records.
Only .csv files. For Excel, save as CSV first.

📥 Bulk Import Fees

📋 How to use:
1. Click "⬇ Download Template" to get the sample CSV.
2. Required columns: rollNo, month, amount, dueDate.
3. month example: April 2025. dueDate format: YYYY-MM-DD.
4. Students must already exist in the system (matched by Roll No).
5. Supports 5000+ records in one import.
Only .csv files.
